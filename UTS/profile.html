Profile

Nama saya adalah Aldiansa Kautsar Ramadhani. Saya Dilahirkan pada 28 Desember 1998 di Malang. Orang tua saya bernama Adjar Sulistyono dan Imaroh Ratna Mufidah. Saya merupakan anak pertama dari tiga bersaudara. Setelah memasuki usia sekolah dasar saya bersekolah di SDI Alma'arif 02 Singosari Malang, saya melanjutkan pendidikan saya di MTsN Lawang(MtsN 3 Malang) dan juga SMKN 2 Singosari.
